## How to View

Open in browser:
- **Hub:** `voice-to-minister-papanasam/index.html`
- **Demo:** `voice-to-minister-papanasam/demo/index.html`

Or run a local server:
```bash
cd voice-to-minister-papanasam
python3 -m http.server 8080
```
Then open: http://localhost:8080
